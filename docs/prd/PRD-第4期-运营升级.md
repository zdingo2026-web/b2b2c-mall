## 2. 功能清单

详见 `docs/04-第四期-运营升级.md` 第三节。

### 2.1 数据看板指标体系

| 指标 | 计算公式 | 更新频率 |
|------|---------|---------|
| GMV | 当日有效订单金额总和 | 实时 |
| UV | 当日独立访客数 | 实时 |
| PV | 当日页面浏览量 | 实时 |
| 转化率 | 支付订单数 / UV | 实时 |
| 客单价 | GMV / 支付订单数 | 实时 |
| 复购率 | 二次购买用户数 / 总购买用户数 | 每日 |
| 新增会员 | 当日新注册会员数 | 实时 |
| 热销TOP10 | 商品销量排行 | 每小时 |

### 2.2 搜索增强规格

| 功能 | 规格 |
|------|------|
| 拼音搜索 | 全拼+首字母，如"shouji"/"sj"匹配"手机" |
| 智能分词 | IK分词器，支持同义词配置 |
| 搜索纠错 | 编辑距离算法，如"手几"→"手机" |
| 品牌筛选 | 搜索结果聚合品牌列表 |
| 价格区间 | 搜索结果聚合价格分布，自动生成区间 |

### 2.3 IM客服功能规格

| 功能 | 规格 |
|------|------|
| 消息类型 | 文字/图片/商品卡片/订单卡片 |
| 连接方式 | WebSocket长连接 |
| 客服分配 | 轮询分配/手动转接 |
| 离线处理 | 客服离线→留言模式，上线后查看 |
| 历史消息 | MongoDB存储，可搜索 |
| 消息存储 | 保留6个月 |

### 2.4 消息通知场景映射

| 场景 | 短信 | 邮件 | 站内信 |
|------|------|------|--------|
| 商家审核通过 | ✅ | ✅ | ✅ |
| 提现审核通过 | ✅ | - | ✅ |
| 订单发货 | ✅ | - | ✅ |
| 提货码 | ✅ | - | ✅ |
| 降价通知 | - | ✅ | ✅ |
| 密码找回 | ✅ | ✅ | - |
