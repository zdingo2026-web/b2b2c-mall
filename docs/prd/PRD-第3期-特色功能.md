## 1. 概述

### 1.1 本期目标

实现功能列表中的差异化特色功能，包括拍卖、夺宝奇兵、超值礼包、预售、社区团购、储值卡/充值卡、实体门店、白条分期。

### 1.2 核心价值

- 差异化功能提升平台竞争力
- 社区团购降低获客成本
- 白条/储值卡提升客单价和复购率

### 1.3 依赖前置条件

- 第一期MVP已完成
- 第二期营销引擎已就绪（拍卖/夺宝可复用营销活动框架）

## 2. 功能清单

详见 `docs/03-第三期-特色功能.md` 第三节，此处补充关键业务规则。

### 2.1 拍卖规则

| 规则项 | 说明 |
|--------|------|
| 起拍价 | 商户设置最低起拍价格 |
| 加价幅度 | 每次出价至少增加的金额 |
| 一口价 | 设置可直接成交的价格，出价达到即成交 |
| 保证金 | 参与竞拍需缴纳，成交后抵扣/未成交退还 |
| 出价有效期 | 出价后不可撤回 |
| 成交判定 | 拍卖结束时最高价者成交 |

### 2.2 夺宝奇兵规则

| 规则项 | 说明 |
|--------|------|
| 出价范围 | 最低价~最高价区间，出价必须为整数 |
| 唯一最低价 | 所有出价中只出现一次的最低价格即为中标价 |
| 重复出价 | 同一用户可多次出价 |
| 活动结束 | 达到活动时间或出价次数上限 |
| 中奖通知 | 站内信+短信通知中奖用户 |

### 2.3 预售规则

| 规则项 | 说明 |
|--------|------|
| 定金比例 | 可设置定金占商品价格的比例（如10%/20%） |
| 定金可退 | 默认不可退（可配置） |
| 尾款期 | 定金支付后N天内需支付尾款 |
| 尾款逾期 | 超时未付尾款，订单取消，定金不退 |
| 预订人数 | 详情页实时显示已预订人数 |

### 2.4 社区团购规则

| 规则项 | 说明 |
|--------|------|
| 团长佣金 | 按订单金额百分比结算（可设置） |
| 团长招募 | 发布招募令，用户申请→平台审核 |
| 社区绑定 | 团长绑定一个社区（地理位置） |
| 商品来源 | 从普通商品中选择导入，独立定价 |
| 配送方式 | 统一配送至团长处，居民自提 |

### 2.5 白条分期规则

| 规则项 | 说明 |
|--------|------|
| 额度评估 | 基于消费记录/信用分/还款历史 |
| 分期期数 | 3/6/9/12期（可配置） |
| 分期费率 | 每期费率可配置（如0.5%/期） |
| 还款日 | 每月固定还款日 |
| 逾期处理 | 逾期罚息+降额+冻结 |
