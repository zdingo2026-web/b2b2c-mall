## 二、平台管理端页面设计

### 2.1 优惠券模板列表

**路由：** `/platform/promotion/coupon-list`

**页面功能：**
- 查看全平台优惠券模板列表（含平台券和商家券）
- 按条件筛选优惠券模板
- 新增/编辑/启用/禁用/删除优惠券模板
- 查看领取统计

**页面布局：**

```
+----------------------------------------------------------+
| 面包屑: 营销管理 > 优惠券模板列表                            |
+----------------------------------------------------------+
| 筛选区                                                     |
| [券名称] [券类型 v] [发放主体 v] [状态 v] [有效期 v]         |
| [搜索] [重置]                         [+ 新增优惠券]        |
+----------------------------------------------------------+
| 表格区                                                     |
| ID | 券名称 | 券类型 | 面值/折扣 | 门槛 | 发放主体 |        |
| 总量 | 剩余 | 限领 | 有效期 | 状态 | 操作                    |
+----------------------------------------------------------+
| 分页: 共X条 < 1 2 3 ... >                                 |
+----------------------------------------------------------+
```

**列表字段：**

| 字段 | 列宽 | 排序 | 筛选 | 说明 |
|------|------|------|------|------|
| ID | 80px | 支持 | - | 模板ID |
| 券名称 | min-width 150px | - | 输入框模糊搜索 | |
| 券类型 | 100px | - | 下拉：满减/满折/满赠/无门槛 | Tag颜色区分：满减蓝、满折绿、满赠橙、无门槛紫 |
| 面值/折扣 | 120px | - | - | 满减显示"减X元"，满折显示"X折"，满赠显示"赠品"，无门槛显示"减X元" |
| 使用门槛 | 100px | - | - | "满X元"或"无门槛" |
| 发放主体 | 100px | - | 下拉：平台/商家 | 平台显示蓝色Tag，商家显示灰色Tag + 商家名称 |
| 发放总量 | 80px | 支持 | - | |
| 剩余数量 | 80px | - | - | 剩余=0时红色加粗 |
| 每人限领 | 80px | - | - | |
| 有效期 | 180px | - | 下拉：未开始/进行中/已过期 | 固定时间段显示日期范围，领取后N天显示"领取后X天有效" |
| 状态 | 80px | - | 下拉：启用/禁用/已结束 | StatusTag组件 |
| 操作 | 200px | - | - | 查看/编辑/启用(禁用)/删除 |

**按钮操作：**

| 按钮 | 触发条件 | 操作结果 |
|------|---------|---------|
| 新增优惠券 | 始终可见 | 跳转到优惠券模板详情页(新增模式) |
| 搜索 | 点击 | 按筛选条件重新查询列表 |
| 重置 | 点击 | 清空所有筛选条件 |
| 查看 | 每行 | 打开详情抽屉(只读) |
| 编辑 | 每行(status!=已结束) | 跳转到优惠券模板详情页(编辑模式) |
| 启用/禁用 | 每行 | 确认弹窗 -> 切换status字段 |
| 删除 | 每行(remainCount=totalCount) | 二次确认弹窗 -> 逻辑删除 |

**批量操作：**
- 批量启用：选中多条status=0的记录 -> 确认弹窗 -> 批量更新
- 批量禁用：选中多条status=1的记录 -> 确认弹窗 -> 批量更新
