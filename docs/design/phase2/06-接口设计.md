## 一、接口规范

### 1.1 通用约定

| 项目 | 规范 |
|------|------|
| 统一响应 | `R<T>`，code=0成功，非0失败 |
| 分页响应 | `R<PageResult<T>>`，PageResult含list/total/page/limit |
| 分页参数 | `@RequestParam page=1, limit=10`，最大100 |
| 版本前缀 | `/api/v1/` |
| 角色分区 | `platform`(平台) / `merchant`(商户) / `member`(C端) |
| 认证方式 | `Authorization: Bearer {JWT}` |
| 参数校验 | `@Validated @RequestBody` 复杂对象，`@RequestParam` 简单参数 |
| 金额单位 | 元，`BigDecimal`，2位小数 |
| 时间格式 | `yyyy-MM-dd HH:mm:ss` |
| 百分比 | 直接传数值，如10.00表示10% |

### 1.2 HTTP方法规则

| 方法 | 用途 | 示例 |
|------|------|------|
| GET | 查询/列表/详情 | GET /list, GET /{id} |
| POST | 创建/提交/操作 | POST /create, POST /claim |
| PUT | 更新/状态变更 | PUT /update/{id}, PUT /{id}/enable |
| DELETE | 删除 | DELETE /{id} |

### 1.3 二期错误码扩展

| 范围 | 模块 | 示例 |
|------|------|------|
| 10xxx | 优惠券 | 10001-券已领完, 10002-已达限领, 10003-券不可用 |
| 11xxx | 秒杀 | 11001-活动未开始, 11002-活动已结束, 11003-商品已售罄, 11004-已达限购, 11005-令牌无效 |
| 12xxx | 拼团 | 12001-拼团已满, 12002-拼团已过期, 12003-已达开团上限 |
| 13xxx | 积分 | 13001-积分不足, 13002-库存不足, 13003-今日已签到 |
| 14xxx | 分销 | 14001-佣金不足, 14002-未达提现门槛, 14003-今日已提现 |
| 15xxx | 装修 | 15001-组件类型非法, 15002-XSS内容拒绝 |

**CouponTemplateCreateDTO**:

```java
public class CouponTemplateCreateDTO {
    @NotBlank(message = "券名称不能为空")
    @Length(min = 2, max = 50, message = "券名称2-50字符")
    private String couponName;
    
    @NotNull(message = "券类型不能为空")
    private Integer couponType;  // 1-满减 2-满折 3-满赠 4-无门槛减
    
    @NotNull(message = "面值不能为空")
    @DecimalMin(value = "0.01", message = "面值必须大于0")
    private BigDecimal couponValue;
    
    private BigDecimal minAmount;  // 使用门槛(满减/满折必填)
    private BigDecimal maxDiscount; // 折扣上限(满折时)
    private Long giftSkuId;        // 赠品SKU(满赠时)
    
    @NotNull(message = "发放总量不能为空")
    @Min(value = 1, message = "发放总量>=1")
    private Integer totalCount;
    
    @NotNull(message = "每人限领不能为空")
    @Min(value = 1, message = "每人限领>=1")
    private Integer limitPerUser;
    
    @NotNull(message = "适用范围不能为空")
    private Integer applyScope;  // 0-全店 1-指定分类 2-指定商品
    
    private String applyCategoryIds;  // JSON数组
    private String applySpuIds;       // JSON数组
    
    @NotNull(message = "有效期类型不能为空")
    private Integer validType;  // 1-固定时间段 2-领取后N天
    
    private LocalDateTime validStartTime;
    private LocalDateTime validEndTime;
    private Integer validDays;
    
    private Boolean canStackSeckill = false;
}
```
